# NEXT FRAME v2

next-frame-v2-production.up.railway.app

一个正在锐意 **vibe coding** 中的番剧推荐系统。🚧

目前项目基于 Bangumi 的匿名用户交互数据，使用 **BPR（Bayesian Personalized Ranking）** 训练用户与番剧的隐向量，并据此做一个简单的个性化推荐流程：

```text
Bangumi 用户行为数据
        ↓
BPR 矩阵分解训练
        ↓
用户向量 / 番剧向量
        ↓
候选召回与排序
        ↓
推荐用户可能喜欢的下一部番
```

## 现在做到哪了

- 使用 Bangumi 数据训练 BPR 推荐模型
- 根据用户看过、想看、评分等行为学习长期偏好
- 基于番剧隐向量进行相似作品召回
- 支持简单的推荐排序与离线指标评估
- 持续尝试多兴趣画像、内容特征、评论理解和冷启动方案

## 为什么做这个

很多推荐系统擅长告诉你“大家都在看什么”，但不一定知道“你下一部真正想看的是什么”。NEXT FRAME 想做的是：在协同过滤的基础上，逐步理解用户喜欢一部作品的原因，并把这种偏好迁移到还没看过的作品上。

当前版本仍然很早，BPR 主要负责学习：

> 喜欢这些番的人，通常还会喜欢什么？

后续会继续探索：

- 用户长期兴趣与近期观看意愿
- 多兴趣簇，而不是单一平均画像
- 番剧评论中的隐性体验特征
- 新番与长尾作品的内容冷启动
- 召回、粗排、精排和多样性重排
- 更可解释的推荐理由
